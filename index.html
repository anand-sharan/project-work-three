    • Summary: USA Elections

    The purpose of this project was to Analysis of US Presidential Results of 2016 and 2020 to find trends in voting patterns and identify possible reasons for those trends.

    After Web Scraping the dataset, we used HTML, SVG, D3, Chloropeth JS library and Anime JS library to plot various aspects of the USA Election results. Factors we looked at included: Voter Turnout, Household Income, Median Age, and Poverty. This site provides the visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations observed.
